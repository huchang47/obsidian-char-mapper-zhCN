---

## � 项目介绍

铁拳输入法是一款专为《铁拳》系列格斗游戏玩家打造的 Windows 系统级输入工具。它可以将普通的键盘输入（如 WASD）实时转换为铁拳游戏社区通用的符号表示法（如 ⬆⬅⬇➡、❶❷❸❹ 等），方便玩家在撰写攻略、讨论招式、记录连招时使用标准符号。

### 核心特性

- **系统级输入**：在任何应用程序中均可使用（QQ、微信、记事本、浏览器等）
- **双模式设计**：铁拳模式专为铁拳设计，方向键模式可以为其他格斗游戏使用
- **大小写敏感**：支持 Shift+方向键输出横移符号（⇈/⇊），完美还原铁拳招式表示
- **热键切换**：默认 F8 快速开启/关闭，不影响正常输入
- **高度可定制**：可自定义所有按键映射，创建属于自己的输入方案

## �🚀 使用方法

### 1. 启动输入法
- 双击运行 `TekkenInputMethod.exe`
- 程序会在系统托盘显示图标

### 2. 激活输入法
- **快捷键**：按 `F8` 快速开启/关闭输入法（默认热键，可在配置中修改，注意不要使用被转换的按键）
- 点击主界面的"激活输入法"按钮
- 或通过系统托盘菜单选择"激活"
- 激活后，输入WASD等按键会自动转换为对应符号

### 3. 输入映射

| S | ⬇ | 下方向 |
| Shift+S | ⇊ | 外横移 |
| D | ➡ | 右方向 |
| Q | ↖ | 左上方向 |
| E | ↗ | 右上方向 |
| Z | ↙ | 左下方向 |
| C | ↘ | 右下方向 |
| J | ❶ | 按钮1 |
| I | ❷ | 按钮2 |
| K | ❸ | 按钮3 |
| L | ❹ | 按钮4 |
| U | ❶✚❷ | 组合按钮1+2 |

| Z | ↙ | 左下方向 |

### 4. 两种模式的区别

**铁拳模式**：
- 专为铁拳游戏设计
- 包含完整的方向键和按钮映射
- 有组合按钮、特殊符号等功能
- 支持大小写敏感（Shift+方向键输出横移符号）
- 适合游戏出招记录

**方向键模式**：
- 简洁的方向键映射
- 只包含基本的方向键和斜方向
- 适合一般用途的箭头输入
- 界面更简洁

### 5. 配置输入法
- 点击"配置"按钮打开配置界面
- **按键映射标签页**：可以自定义按键映射，支持大小写敏感（如 `w` 和 `W` 可映射不同符号）
- **预设模式标签页**：
  - 显示详细的模式说明
  - 切换铁拳模式或方向键模式
  - 恢复默认设置
- **系统设置标签页**：可以设置开机自启、修改热键等选项

### 6. 系统托盘功能
- 右键点击托盘图标
- 快速激活/停用输入法
- 打开配置界面
- 退出程序

### 7. 停用输入法
- 再次按 `F8` 热键（或你自定义的热键）
- 点击"激活输入法"按钮
- 或通过系统托盘菜单选择"停用"

## 🔧 技术改进

### 按键拦截机制
- 使用低级键盘钩子拦截按键事件
- 成功映射后阻止原始按键传递
- 确保只输出转换后的字符

### 字符兼容性
- 使用标准Unicode字符，需支持Emoji字符集
- 在QQ、微信等应用中正常显示
- 斜方向符号采用通用编码

## 🗑️ 卸载方法

1. 关闭输入法程序
2. 删除配置目录 `%APPDATA%\TekkenInputMethod`
3. 如需删除开机自启，请在配置中取消勾选"开机自启"

## ⚠️ 已知限制

- QQ中的斜方向符号（↖↗↙↘）在某些情况下可能被转义，这是QQ的富文本处理机制导致的，输入法已使用标准Unicode字符，但应用端的显示处理不在输入法控制范围内
