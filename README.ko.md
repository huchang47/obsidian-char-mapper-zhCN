Windows 시스템 수준의 철권 입력기. 키보드 입력을 실시간으로 철권 스타일의 방향키와 버튼 기호로 변환합니다.

---

## 📖 프로젝트 소개

철권 입력기는 철권 시리즈 격투 게임 플레이어를 위해 특별히 설계된 Windows 시스템 수준의 입력 도구입니다. 표준 키보드 입력(WASD 등)을 철권 커뮤니티 표준 기호 표기법(⬆⬅⬇➡, ❶❷❸❹ 등)으로 실시간 변환하여 플레이어가 공략을 작성하고, 기술을 논의하고, 콤보를 기록할 때 표준 기호를 사용할 수 있도록 합니다.

### 핵심 기능

- **시스템 수준 입력**: 모든 애플리케이션에서 사용 가능(QQ, WeChat, 메모장, 브라우저 등)
- **듀얼 모드 설계**: 철권 모드는 게임용, 화살표 모드는 일상용
- **대소문자 구분**: Shift+방향키로 옆이동 기호(⇈/⇊) 출력
- **단축키 전환**: 기본 F8로 빠르게 켜기/끄기
- **고도로 사용자 정의 가능**: 모든 키 매핑을 사용자 정의 가능

---

## 🚀 사용 방법

### 1. 실행
- `TekkenInputMethod.exe`를 더블클릭
- 프로그램은 시스템 트레이에서 실행됩니다

### 2. 활성화
- **단축키**: `F8`을 눌러 켜기/끄기 전환(기본값, 사용자 정의 가능)
- 메인 창의 "활성화" 버튼 클릭
- 시스템 트레이 메뉴 사용

### 3. 입력 매핑

#### 철권 모드(기본):

| 키 | 출력 | 설명 |
|------|------|------|
| W | ⬆ | 위 |
| Shift+W | ⇈ | 안쪽 옆이동 |
| A | ⬅ | 왼쪽 |
| S | ⬇ | 아래 |
| Shift+S | ⇊ | 바깥쪽 옆이동 |
| D | ➡ | 오른쪽 |
| Q | ↖ | 왼쪽 위 대각선 |
| E | ↗ | 오른쪽 위 대각선 |
| Z | ↙ | 왼쪽 아래 대각선 |
| C | ↘ | 오른쪽 아래 대각선 |
| J | ❶ | 버튼 1(왼쪽 펀치) |
| I | ❷ | 버튼 2(오른쪽 펀치) |
| K | ❸ | 버튼 3(왼쪽 킥) |
| L | ❹ | 버튼 4(오른쪽 킥) |
| U | ❶✚❷ | 콤보 버튼 1+2 |
| O | ❸✚❹ | 콤보 버튼 3+4 |
| Space | ▸ | 커넥터 |
| N | ✩ | 중립 |
| X | ⇩ | 앉기 |
| H | Ⓗ | H 기능 |
| R | Ⓡ | R 기능 |
| T | Ⓣ | T 기능 |
| B | Ⓑ | B 기능 |

#### 화살표 키 모드(간단 모드):

| 키 | 출력 |
|------|------|
| W/A/S/D | ⬆/⬅/⬇/➡ |
| Q/E/Z/C | ↖/↗/↙/↘ |

### 4. 모드 차이점

**철권 모드**:
- 철권 게임을 위해 특별히 설계
- 완전한 방향키와 버튼 매핑 포함
- 콤보 버튼, 특수 기호 등의 기능
- 대소문자 구분 지원(Shift+방향으로 옆이동 기호)
- 게임 기술과 콤보 기록에 이상적

**화살표 키 모드**:
- 간단한 방향키 매핑
- 기본 방향과 대각선만 포함
- 일반적인 화살표 입력 요구에 적합
- 깔끔한 인터페이스

### 5. 설정
- "설정" 버튼을 클릭하여 설정 창 열기
- **키 매핑 탭**: 키 매핑 사용자 정의(대소문자 구분 지원)
- **프리셋 모드 탭**:
  - 자세한 모드 설명 보기
  - 원클릭으로 철권 모드와 화살표 모드 전환
  - 기본 설정으로 복원
- **시스템 설정 탭**: Windows 시작 시 실행, 단축키 사용자 정의 등

### 6. 시스템 트레이 기능
- 트레이 아이콘을 오른쪽 클릭
- 입력기를 빠르게 활성화/비활성화
- 설정 창 열기
- 프로그램 종료

### 7. 비활성화
- `F8` 단축키를 다시 누르기(또는 사용자 정의한 단축키)
- "활성화" 버튼을 다시 클릭
- 시스템 트레이 메뉴에서 "비활성화" 선택

## 🔧 기술적 개선

### 키 가로채기 메커니즘
- 저수준 키보드 후크를 사용하여 키 이벤트 가로채기
- 매핑 성공 후 원래 키 전송 차단
- 변환된 문자만 출력되도록 보장

### 문자 호환성
- 표준 Unicode 문자 사용(Emoji 지원 필요)
- QQ, WeChat 및 기타 애플리케이션에서 올바르게 표시
- 대각선 기호는 범용 인코딩 사용

---

## 📝 주의사항

1. **관리자 권한**: 일부 기능에는 관리자 권한이 필요할 수 있습니다
2. **백신 소프트웨어**: 백신 소프트웨어에 의해 차단된 경우 화이트리스트에 추가
3. **호환성**: Windows 10/11 지원
4. **로그 파일**: 로그는 `%APPDATA%\TekkenInputMethod\Logs` 디렉토리에 저장

---

## ⚙️ 설정 파일

설정 파일 위치: `%APPDATA%\TekkenInputMethod\config.json`

이 파일을 수동으로 편집하여 매핑 관계를 사용자 정의할 수 있습니다.

---

## 🗑️ 제거 방법

1. 입력기 프로그램 종료
2. 설정 디렉토리 `%APPDATA%\TekkenInputMethod` 삭제
3. Windows 시작 시 실행을 제거하려면 설정에서 "Windows 시작 시 실행" 체크 해제

---

## 🆘 기술 지원

문제가 발생하면 로그 파일을 확인하세요.

## ⚠️ 알려진 제한사항

- QQ의 대각선 기호(↖↗↙↘)가 특정 상황에서 이스케이프될 수 있습니다. 이는 QQ의 리치 텍스트 처리 메커니즘으로 인한 것입니다. 입력기는 표준 Unicode 문자를 사용하지만, 애플리케이션 측의 표시 처리는 입력기의 제어 범위를 벗어납니다.
